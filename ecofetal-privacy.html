2) Quais dados coletamos

O EcoFetal Pro não exige login e, por padrão, não coleta dados pessoais identificáveis (como nome, e-mail, CPF, endereço) para funcionar.

Caso você insira informações no app (por exemplo, anotações, medidas, cálculos, parâmetros clínicos), esses dados são usados apenas para a funcionalidade do aplicativo.

3) Onde os dados ficam

Sempre que possível, as informações inseridas no app permanecem no próprio dispositivo do usuário.

4) Compartilhamento de dados

Não vendemos nem compartilhamos dados pessoais com terceiros. Podemos compartilhar informações apenas se exigido por lei ou para cumprir obrigações legais.

5) Compras e assinaturas

Caso o app ofereça compras/assinaturas, o processamento é feito pela Apple (App Store). Não recebemos dados completos do seu cartão. Podemos receber apenas informações básicas da transação (ex.: status da assinatura) para liberar recursos dentro do app.

6) Permissões do dispositivo

O app pode solicitar permissões do dispositivo apenas quando necessário para recursos específicos (ex.: arquivos/armazenamento para exportar conteúdos). Você pode gerenciar permissões nas configurações do iOS.

7) Segurança

Adotamos medidas razoáveis para proteger as informações tratadas pelo aplicativo. Ainda assim, nenhum sistema é 100% seguro.

8) Público-alvo

O EcoFetal Pro é destinado a profissionais/estudantes da área de saúde para apoio educacional e de produtividade. O aplicativo não substitui avaliação médica, diagnóstico ou conduta clínica.

9) Seus direitos

Se você tiver dúvidas, solicitações ou quiser pedir exclusão de dados (quando aplicável), entre em contato pelo e-mail: pierrepaiva@hotmail.com.
